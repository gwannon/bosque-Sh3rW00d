# Bosque-Sh3rW00d

Bosque Sh3rW00d es una ambientación CyberPunk para Savage Worlds Edición Explorador en un mundo extrañamente parecido a las historias de Robin Hood o Ivanhoe.

[Página web](https://bosque.gwannon.com)
